# leetcode

1. create env

```
python3 -m venv env
source env/bin/activate
```

2. install requirements

```
pip install -r requirements.txt
```

3. to test solution in folder "Some Problem"

with pytest

```
cd Some\ Problem
pytest -v test_solution.py
```

execute script with python

```
cd cd Some\ Problem
python3 test_solution.py
```
